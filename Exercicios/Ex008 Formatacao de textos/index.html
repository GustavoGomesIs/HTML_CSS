Principais formatações

Negrito / Destaque

Nesta mensagem tem um texto em Negrito usando a tag <b> (Não semantico)

Nesta mensagem tem um destaque usando a tag <strong> (Semantico)

Italico / Ênfase

Nesta mensagem tem um texto em Intalico usando a tag <i> (Não semantico)

Nesta mensagem tem um texto Ênfase usando a tag <em> (Semantico)

Texto Marcado/ Estilo marca texto

Podemos usar uma frase com marca texto usando a tag <mark>

E tmb pode trocar a cor do marca texto usando a tag <mark style>

Texto Grande e Pequeno

O texto grande se faz com CSS3

Nesta mensagem tem uma frase pequena usando a tag <small>

Texto Deletado

Texto deletado seria um texto excluido, mas ainda deve ser lido, mas não considerado usando a tag <dell>

Texto Inserido

Podemos marcar um texto inserido para dar uma ênfase e indicar que ele foi adicionado usando a tag <ins>

E tambem pode se usar a tag <u> para fazer o sublinhado (não sematica)

Texto Sobrescrito

Usado para inserir coisas como x20+3 usando a tag <sup>

TextoSubscrito

Usado para inserir coisas como H2O usando a tag <sup>
